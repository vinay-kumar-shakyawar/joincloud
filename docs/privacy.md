# Privacy Policy

JoinCloud collects anonymous, aggregated operational telemetry to improve reliability.

## What is collected
- App uptime
- Feature usage counts
- Data volume totals (bytes uploaded/downloaded)
- Share counts (public vs LAN)

## What is NOT collected
- File names, paths, extensions, or contents
- IP addresses
- Personal information

Telemetry is anonymous and can be disabled at any time in Settings.
